Microsserviço de Categorização de Produtos (v1)

Download OpenAPI specification:

License: MIT

API responsável pelo gerenciamento de categorias de produtos do sistema Plus. Permite criar e manter categorias, incluindo subcategorias (hierarquia). A associação entre produtos e categorias é responsabilidade do MS de produtos.

Categorias

CRUD de categorias de produtos

Lista categorias

Retorna as categorias de forma paginada, com filtros opcionais.

Authorizations:
bearerAuth
query Parameters
nome
string

Filtra por nome (busca parcial, case-insensitive).

ativo
boolean

Filtra por status (ativas/inativas).

categoriaPaiId
integer <int64>

Lista apenas subcategorias diretas da categoria informada.

page
integer >= 0
Default: 0

Número da página (base 0).

size
integer [ 1 .. 100 ]
Default: 20

Quantidade de itens por página.

Responses

Ambiente local (Docker / dev)

http://localhost:3002/categorias

Via API Gateway (ministack)

http://localhost:3002/categorizacao/categorias

  • 401

Cria uma nova categoria

Apenas usuários com perfil ADMIN podem criar categorias.

Authorizations:
bearerAuth
Request Body schema: application/json
required
nome
required
string [ 2 .. 80 ] characters
descricao
string <= 255 characters
ativo
boolean
Default: true
categoriaPaiId
integer or null <int64>

ID da categoria pai (para subcategorias). Null = categoria raiz.
